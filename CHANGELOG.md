# Changelog

All notable changes to this project will be documented in this file.

This project follows Semantic Versioning:

- Patch versions fix bugs without changing the public API.
- Minor versions add endpoint, type, or behavior features in a backward-compatible way.
- Major versions may change the public API exported from `src/index.ts`.
